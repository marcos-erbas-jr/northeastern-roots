# northeastern-roots
Projeto de Desenvolvimento Back-end do Curso de Análise e Desenvolvimento de Sistemas
Instruções:

1.	Clonar o repositório no link: https://github.com/marcos-erbas-jr/northeastern-roots
2.	Instale as dependências que estão no requirements.txt que está dentro do diretório app/ por meio do: pip install -r requirements.txt
3.	Crie uma nova venv: python -m venv venv
4.	Ative a venv: venv\Scripts\activate
5.	Este passo é  muito importante, rode a seed.py para popular o banco de dados, pois a interface inicial exige a existência de pelo menos uma filial/unidade
